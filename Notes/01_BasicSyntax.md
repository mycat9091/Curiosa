## 标题

使用 `#` 符号创建标题。数量越多，级别越低。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

---

## 文本格式

### 加粗
```markdown
**加粗文本**
__也可以这样写__
```
显示为：**加粗文本**

### 斜体
```markdown
*斜体文本*
_也可以这样写_
```
显示为：*斜体文本*

### 加粗且斜体
```markdown
***加粗且斜体***
___也可以这样写___
```
显示为：***加粗且斜体***

### 删除线
```markdown
~~删除的文本~~
```
显示为：~~删除的文本~~

### 高亮
```markdown
==高亮文本==
```
显示为：==高亮文本==

### 下划线
```markdown
<u>下划线文本</u>
```
显示为：<u>下划线文本</u>

## 列表

### 无序列表
```markdown
- 项目一
- 项目二
- 项目三

* 也可以用星号
+ 或加号
```

### 有序列表
```markdown
1. 第一项
2. 第二项
3. 第三项
```

### 嵌套列表
```markdown
- 主项目
  - 子项目一
  - 子项目二
    - 孙项目
- 主项目二
```

### 混合列表
```markdown
1. 有序项一
   - 无序子项
   - 无序子项
2. 有序项二
```

---

## 分割线

三个或以上的 `-`、`*` 或 `_` 都可以：

```markdown
---

***

___
```

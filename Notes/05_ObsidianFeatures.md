# Obsidian特有功能

[← 回到主目录](00_Index.md)

## 双向链接

### 基本双向链接
最核心的Obsidian功能，链接到同一库中的笔记：

```markdown
[[笔记名称]]
```

**特点：**
- 自动创建反向链接
- 在被链接的笔记中自动显示反向链接
- 支持图形视图展示知识图谱
- 红色链接表示目标笔记不存在，点击可快速创建

### 自定义显示文本
```markdown
[[真实笔记名称|显示的别名]]
```

例如：
```markdown
[[项目计划|查看计划]]
```
链接指向"项目计划"笔记，但显示文本为"查看计划"

### 链接到特定标题
```markdown
[[笔记名称#二级标题]]
[[笔记名称#二级标题|自定义文本]]
```

例如：
```markdown
[[日志#2026-07-07]]
[[会议记录#行动项|本周行动项]]
```

### 包含空格的笔记名称
```markdown
[[文件夹/笔记 名称]]
[[项目/项目A/计划文档]]
```

## 标签

### 基本标签
为笔记添加标签进行分类和查找：

```markdown
#标签
#项目
#待处理
```

### 嵌套标签
使用 `/` 创建分层标签：

```markdown
#项目/项目A
#项目/项目A/任务1
#项目/项目B
#阅读/技术/Python
```

**优势：**
- 标签窗格中显示标签树
- 支持多级分类
- 点击标签可查看所有使用该标签的笔记

### 多个标签
```markdown
#重要 #待处理 #项目/A
```

### 标签特点
- 标签不能包含空格
- 标签前必须有 `#` 符号
- 点击标签自动搜索所有使用该标签的笔记

## 块引用

### 添加块ID
给段落、代码块或列表项添加ID以便精确引用：

```markdown
某些段落文本 ^块ID

- 列表项 ^列表ID

代码块中的内容 ^代码ID
```

块ID必须以 `^` 开头，只能包含字母、数字、破折号。

### 引用特定块
```markdown
[[笔记名称#^块ID]]
[[笔记名称#^块ID|自定义文本]]
```

例如：
```markdown
[[会议记录#^action-items]]
```

## 嵌入（Embed）

### 嵌入整个笔记
在当前笔记中显示另一笔记的全部内容：

```markdown
![[笔记名称]]
```

### 嵌入笔记的特定部分

**嵌入特定标题下的内容：**
```markdown
![[笔记名称#二级标题]]
```

**嵌入特定块：**
```markdown
![[笔记名称#^块ID]]
```

### 嵌入图片
```markdown
![[image.png]]
```

与 `![](image.png)` 的区别：嵌入会显示完整尺寸的图片，且支持预览。

### 嵌入代码块
```markdown
![[代码笔记#^代码块ID]]
```

### 嵌入优点
- 避免重复编写内容
- 内容修改时自动更新
- 便于构建模块化知识库

## 快速操作

### 快速创建链接
在编辑模式下输入 `[[` 然后开始输入，Obsidian会自动提示相关笔记。

### 快速添加标签
输入 `#` 然后输入标签名，Obsidian会自动提示已有标签。

### 反向链接窗格
- 显示所有链接到当前笔记的其他笔记
- 点击可快速导航
- 帮助发现关联的笔记

### 图形视图
- 可视化笔记之间的链接关系
- 右键点击笔记可快速打开
- 支持过滤和搜索

## 别名和元数据

### YAML Front Matter
在笔记顶部添加元数据（可选）：

```markdown
---
tags: [标签1, 标签2]
aliases: [别名1, 别名2]
date: 2026-07-07
---

笔记内容从这里开始
```

### 别名用途
通过别名可以用不同名称引用同一笔记：

```markdown
[[真实笔记名|别名]]
```

# 引用与任务

[← 回到主目录](00_Index.md)

## 引用

### 基本引用
```markdown
> 这是一条引用文本
```
显示为：
> 这是一条引用文本

### 多行引用
```markdown
> 这是第一行
> 这是第二行
```

### 嵌套引用
```markdown
> 这是外层引用
>> 这是嵌套引用
>>> 这是更深层的嵌套
```

### 引用中的其他元素
```markdown
> **加粗文本**
> 
> - 列表项一
> - 列表项二
```

## 任务列表

### 基本任务列表
使用 `- [ ]` 创建未完成任务，`- [x]` 创建已完成任务：

```markdown
- [ ] 未完成的任务
- [x] 已完成的任务
- [ ] 另一个任务
```

显示为：
- [ ] 未完成的任务
- [x] 已完成的任务
- [ ] 另一个任务

在Obsidian中，点击复选框可快速切换任务状态。

### 嵌套任务列表
```markdown
- [ ] 主任务
  - [ ] 子任务一
  - [x] 子任务二
- [x] 已完成的主任务
  - [x] 子任务已完成
```

### 在有序列表中使用
```markdown
1. [ ] 有序任务一
2. [x] 有序任务二
3. [ ] 有序任务三
```

## 脚注

### 添加脚注
```markdown
这是一个带脚注的文本[^1]

[^1]: 这是脚注的内容
```

显示为：
这是一个带脚注的文本[^1]

[^2]: 这是脚注的内容

### 脚注特点
- 脚注可以包含多行内容
- 脚注定义可以在文档任何地方
- Obsidian会自动生成脚注编号

---

## 注释

### 隐藏注释
使用 `%%` 包裹的内容在预览模式下不会显示：

```markdown
%%这是注释，预览时不显示%%
```

### 注释用途
- 添加草稿或待编辑的内容
- 临时隐藏某些文本
- 添加个人笔记或提醒

## HTML支持

### 使用HTML标签
Obsidian支持在Markdown中混用HTML：

```markdown
<u>下划线文本</u>

<div style="color: red;">红色文本</div>

<span style="background-color: yellow;">背景高亮</span>
```

### 常用HTML标签
```markdown
<br>          换行
<u>...</u>    下划线
<mark>...</mark> 高亮标记
<sup>...</sup> 上标
<sub>...</sub> 下标
```

---

## 转义特殊字符

### 转义Markdown符号
在特殊字符前加反斜杠 `\` 进行转义：

```markdown
\*不是斜体\*
\[不是链接\]
\# 不是标题
\` 不是代码
```

显示为：
\*不是斜体\*
\[不是链接\]
\# 不是标题

### 需要转义的特殊字符
```
\ ` * _ { } [ ] ( ) # + - . !
```

```markdown
下标：H~2~O (通过~表示下标)

```

```markdown
内联公式：$E=mc^2$

$$
\int_0^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
$$
```
